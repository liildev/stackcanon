# StackCanon 0.1.0 Announcement

`StackCanon 0.1.0` is the first public release of `stackcn`.

It gives existing JavaScript and TypeScript projects a stack-aware setup layer for framework policy, quality-tool integration, AI-facing project context, and official docs sync.

What ships in `0.1.0`:

- support for Next.js, Nuxt, React + Vite, Vue + Vite, TanStack Start, TanStack Query, NestJS, Express, and Fastify
- Ultracite, Biome, and Oxlint quality-provider setup
- canonical `ai/` generation with derived `AGENTS.md`, `CLAUDE.md`, and `.ai-rulez/`
- docs sync into `.stackcn/sources/`
- bundled npm CLI packaging with tarball install smoke coverage

This release is aimed at teams that want one CLI to standardize project governance, AI instructions, and stack-aware config baselines without rewriting the whole repo.
