Release the first public StackCanon CLI.

This release ships:
- bundled `stackcn` distribution
- framework adapters for Next, Nuxt, Vite React, Vite Vue, TanStack Start, TanStack Query, Nest, Express, and Fastify
- quality adapters for Ultracite, Biome, and Oxlint
- canonical `ai/` generation plus derived `.ai-rulez/`, `AGENTS.md`, and `CLAUDE.md`
- official docs sync into `.stackcn/sources/`
- release and publish automation for the public CLI package
